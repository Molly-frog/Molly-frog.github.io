How to Terminal

What Is?

  • Where you type commans to the comp.
  • Don't run a command just because you can

Command Syntax

  • To every command there are 3 parts
    • Utility (Always First)
    • Flags
    • Arguments

Example:

ls -l ~/Desktop

This is what your computer will give back to you(Or at least for me):

total 0
ls:/Users/1015966/Desktop:Operation not permitted
  • ls = Utility
    • Most commands = 1 utility
  • -l = Flag
    • Alters how utility operates
    • Example: Options/Preferences
    • Start with 1 or 2 dash (-)
  • ~/Desktop = Arguments
    • Used when arg. needs to know exactly what you want
    • The # of arg. depends on utility

Moving Around

ls(List Directory)
  • Tells you what names/files are in your directory
cd(Change Directory)
  • Lets you change directory
pwd(Print Working Directory)
  • Lets you see what directory your working in
